📌 Project Overview

This project is a Many-to-Many Thread Model Simulator built using Flask (Backend) and Frontend UI (HTML, CSS, JS).

It demonstrates how multiple user-level threads are mapped to multiple kernel-level threads, simulating real-world Operating System thread management.

🎯 Objective
To simulate the Many-to-Many Thread Model
To visualize thread scheduling and execution
To understand concepts like:
User Threads vs Kernel Threads
Thread Mapping
Scheduling Algorithms

⚙️ Features
✅ Simulation of Many-to-Many Thread Model
✅ Implementation of scheduling algorithms:
FCFS (First Come First Serve)
SJF (Shortest Job First)
Round Robin
✅ Interactive UI for input and visualization
✅ Backend logic using Flask APIs
✅ Real-time execution flow display

🏗️ Tech Stack
🔹 Frontend
HTML
CSS
JavaScript
🔹 Backend
Python
Flask

🧠 How It Works
User inputs processes/threads from UI
Frontend sends data to Flask backend
Backend applies selected scheduling algorithm
Thread mapping (Many-to-Many) is simulated
Results are displayed as execution timeline
